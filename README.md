# Email Constructor PRO

Email Constructor — это веб-приложение на базе фреймворка Streamlit, предназначенное для быстрой визуальной сборки валидных HTML-писем для email-рассылок. 

Инструмент избавляет маркетологов от необходимости работать с исходным кодом. Пользователь заполняет формы в интерфейсе, а программа автоматически генерирует готовый HTML-код, который можно скопировать и вставить в любую систему рассылок (Unisender, SendSay, MailChimp и др.).

## Основные возможности

*   No-Code интерфейс: Сборка писем с табличной версткой через текстовые поля.
*   Умное форматирование: Автоматическая обработка вводимого текста (превращает **текст** в жирный шрифт, а переносы строк — в списки/буллиты).
*   Интеграция таймеров: В акционных шаблонах реализована функция разрезания HTML-кода для удобной вставки сторонних скриптов (таймеров обратного отсчета).
*   Синхронизация данных: Единый блок настройки контактных данных (Email, Телефон, Ссылки), которые автоматически подтягиваются в шапку и подвал письма.
*   Адаптивный UI: Поддержка темной и светлой темы интерфейса.

## Поддерживаемые шаблоны

Конструктор включает 5 готовых бизнес-сценариев для B2B-сегмента:

1.  Спецпредложение (Promo): Шаблон для акций. Включает блок с персональными ценами, баннер со скидкой и место под счетчик времени.
2.  Поступление (Stock): Информационное письмо об обновлении склада. Содержит блоки с перечислением ГОСТов, размеров и сетку товаров.
3.  Отгрузки (Cases): Формат кейса. Позволяет оформить задачу клиента, процесс реализации, показать статистику и прикрепить фото товаров.
4.  Услуги (Services): Презентация производственных возможностей с кнопками призыва к действию.
5.  Экспертное (Expert): Шаблон для контент-маркетинга. Вывод полезной статьи с привязкой к конкретному менеджеру компании.

## Технический стек

*   Язык: Python 3.9+
*   Фреймворк: Streamlit
*   Верстка писем: Табличный HTML/CSS (максимальная кросс-клиентская совместимость).

## Установка и запуск

1. Скачайте репозиторий на свой компьютер.
2. Убедитесь, что у вас установлен Python.
3. Запустите файл START.bat. Скрипт автоматически установит необходимые библиотеки и откроет программу в браузере по локальному адресу.

Альтернативный запуск через консоль:
pip install streamlit
streamlit run app.py

## Инструкция по использованию

1. Откройте приложение.
2. Выберите нужный тип рассылки на главном экране.
3. Заполните данные по вкладкам (Контакты, Баннер, Тексты, Блоки).
4. Нажмите кнопку "СОБРАТЬ ФИНАЛЬНЫЙ HTML".
5. Проверьте визуальное превью письма.
6. Разверните блок с кодом, скопируйте HTML и вставьте в редактор вашего email-сервиса.
